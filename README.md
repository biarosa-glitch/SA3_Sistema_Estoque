# **Sistema de Controle de Estoque Simplificado (SCES)**
## *Controle de saída e entrada de almoxarifados.*

### O Sistema de Controle de Estoque Simplificado (SCES) serve como uma maneira automatizada de controlar um sistema de uma fábrica, permitindo seu monitoramento e alteração. É uma maneira mais rápida e eficiente (com menor risco de erros) de colaborar com a organização de produtos de uma empresa.
### Esse sistema se inicia com um menu, onde são apresentadas ao usuário 5 opções: 1 - Adicionar produtos aos sistema; 2 -Listar os já existentes; 3 - Buscar dentre os produtos registrados; 4 - Modificar/atualizar a quantidade do estoque; 5 - Verificar estoque mínimo. 
### Caso o usuário deseje adicionar um produto, será necessário informar o ID, nome do produto, quantidade e localização (sendo essa última em colunas nomeadas de A-Z com números de três dígitos: A-001, por exemplo).
### Caso deseje listar os produtos já armazenados no sistema, o menu irá mostrá-los com a mesma estrutura: ID, Nome do Produto, Quantidade e Localização.
### No caso de escolher buscar dentre os produtos registrados, deve informar o ID do produto.
### Para modificar/atualizar a quantidade do produto no estoque, o usuário deve informar o ID correto e a nova quantidade, seja de entrada ou saída. Podendo verificar a mudança ao pedir para listar os materiais.
### Quando o funcionário quiser saber quais materiais deve repor, pode optar por verificar o estoque mínimo, onde terá todas as informações do produto e um destaque para sua quantidade.
### Qualquer digitação errada será informada pelo sistema para refazer. Sendo a 6º opção do menu a saída.
